## How to Use This Document

1. **During PI Planning:** Capture ideas as they come up
2. **Rough Size:** Assign Epic/Feature/Story category based on scope
3. **Discuss:** Add context about why, value, dependencies
4. **Graduate:** When ready, use Planning Assistant to create formal epic/feature/story docs
5. **Archive:** Move completed brainstorming sessions to bottom, keep document fresh

**Integration with Planning Assistant:**
- Brainstorm here ‚Üí Create epic with `/epic create`
- Feature ideas ‚Üí Break down with `/feature create`
- Story ideas ‚Üí Add to PI with `/story create`
